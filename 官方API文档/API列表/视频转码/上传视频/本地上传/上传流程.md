# 上传流程

参照 [文件上传](https://123yunpan.yuque.com/org-wiki-123yunpan-muaork/cr6ced/uihb5w71iwqvi9dw)，需要注意的是，调用**创建文件**接口`/upload/v1/file/create`时传入`parentFileID`必须是转码空间的文件夹ID

## Body参数差异
| **名称** | **类型** | **是否必填** | **说明** |
| :---: | :---: | :---: | :---: |
| parentFileID | number | 必填 | 父目录id，必须填写转码空间的文件夹ID |
